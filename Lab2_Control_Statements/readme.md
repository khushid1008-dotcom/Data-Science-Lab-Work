# Lab 2: Control Statements in Python

## Objective
The objective of this lab is to understand how control statements work in Python. Control statements allow a program to make decisions and execute different blocks of code based on conditions. They are essential for building logical and dynamic programs.

## Description
In this lab, we explored various control statements used in Python programming. Control statements help control the flow of execution in a program by allowing decisions, repetitions, and conditional logic.

The lab demonstrates how conditions can be evaluated and how loops can be used to execute tasks repeatedly. These concepts are fundamental for developing efficient programs and solving real-world problems.

## Topics Covered
- If Statement
- If-Else Statement
- If-Elif-Else Statement
- For Loop
- While Loop
- Break Statement
- Continue Statement

## Implementation
The Python program in this lab includes examples of decision-making statements and loops. Different conditions are evaluated using if statements, and loops are used to repeat tasks multiple times.

The code demonstrates:
- Checking conditions using if statements
- Executing alternate code using if-else
- Multiple condition checks using if-elif-else
- Iteration using for loops
- Repetition using while loops
- Controlling loop execution using break and continue

## Tools Used
- Python Programming Language
- Jupyter Notebook

## Learning Outcome
After completing this lab, we gain a clear understanding of how control statements work in Python. These concepts are important for writing logical programs, handling conditions, and managing program flow efficiently.
