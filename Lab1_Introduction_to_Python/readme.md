# Lab 1: Introduction to Python

## Objective
The objective of this lab is to understand the basic concepts of Python programming including variables, data types, input/output operations, and simple calculations.

## Description
In this lab, we explored the fundamentals of Python programming. Python is a high-level programming language widely used in data science, machine learning, and software development.

The exercises in this lab focus on writing simple Python programs to understand how Python syntax works and how programs execute step by step.

## Topics Covered
- Python syntax
- Variables and data types
- Input and output statements
- Basic arithmetic operations
- Simple Python programs

## Tools Used
- Python
- Jupyter Notebook

## Learning Outcome
After completing this lab, we gain a basic understanding of Python programming and how to write simple programs using Python.
