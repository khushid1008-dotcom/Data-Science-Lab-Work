# Lab 4: Pandas

## Objective
The objective of this lab is to learn how to use Pandas for data manipulation and analysis.

## Description
Pandas is one of the most important libraries used in data science. It provides powerful data structures like DataFrames and Series which allow easy manipulation of structured data.

In this lab we explored how to load datasets and perform basic data analysis.

## Topics Covered
- Pandas DataFrames
- Reading CSV files
- Data inspection
- Data filtering and selection
- Basic data analysis

## Tools Used
- Python
- Pandas
- Jupyter Notebook

## Learning Outcome
After completing this lab, we understand how to manipulate datasets and perform basic data analysis using Pandas.
