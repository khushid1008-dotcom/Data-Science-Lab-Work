# Lab 6: KNN Classification

## Objective
The objective of this lab is to understand classification algorithms using the K-Nearest Neighbors (KNN) model.

## Description
KNN is a supervised machine learning algorithm used for classification tasks. It classifies data points based on the similarity with neighboring data points.

In this lab we implemented the KNN algorithm to classify data and evaluate the model performance.

## Topics Covered
- Classification in machine learning
- K-Nearest Neighbors algorithm
- Model training
- Prediction and evaluation

## Tools Used
- Python
- Scikit-learn
- Pandas
- NumPy

## Learning Outcome
After completing this lab, we understand how classification models work and how KNN can be used to classify datasets.
