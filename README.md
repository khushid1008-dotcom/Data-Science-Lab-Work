## About

This repository contains my Data Science Lab work completed as part of the Data Science course at Golden Gate University.

The purpose of this repository is to demonstrate fundamental concepts of data science and machine learning through practical laboratory exercises and projects.

The labs include hands-on implementation of Python programming, control statements, numerical computing with NumPy, data manipulation using Pandas, regression analysis, and classification using the K-Nearest Neighbors algorithm.

In addition to the lab exercises, this repository also includes small projects that demonstrate data analysis, visualization, and basic machine learning techniques.

These implementations help in understanding how data science tools and algorithms are used to analyze data and build predictive models.

## Labs

- Lab1 – Introduction to Python
- Lab2 – Control Statements
- Lab3 – NumPy
- Lab4 – Pandas
- Lab5 – Regression Analysis
- Lab6 – KNN Classification
