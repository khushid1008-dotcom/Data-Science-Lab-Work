# Lab 3: NumPy

## Objective
The objective of this lab is to understand the use of NumPy for numerical computing in Python.

## Description
NumPy is a powerful Python library used for numerical operations and scientific computing. It provides support for arrays, matrices, and mathematical operations.

In this lab we worked with NumPy arrays and performed different numerical operations.

## Topics Covered
- Creating NumPy arrays
- Array indexing and slicing
- Mathematical operations on arrays
- Statistical functions
- Multi-dimensional arrays

## Tools Used
- Python
- NumPy
- Jupyter Notebook

## Learning Outcome
After completing this lab, we understand how NumPy helps in performing fast numerical computations in data science.
